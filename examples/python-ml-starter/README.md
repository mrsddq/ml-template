# Python ML

A small Python machine learning starter project with a clear source, data, model, and test layout.

## Structure

```text
src/python_ml/
  pipeline.py
tests/
  test_pipeline.py
data/
  raw/.gitkeep
  processed/.gitkeep
models/.gitkeep
requirements.txt
```

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
pytest
```
