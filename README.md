# ML Template

A reusable Python machine learning project template with source, tests, experiment folders, packaging files, and example starter layouts.

## Structure

```text
src/
tests/
experiments/
examples/
  python-ml-starter/
```

## Included Examples

- `examples/python-ml-starter`: merged from the standalone `python-ml` repo as a simple baseline ML project layout.

## Status

This is the preferred home for generic Python ML starter work. Keep project-specific ML/CV repos separate when they have a clear applied purpose.
